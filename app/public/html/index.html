crypto-api.info

get public api for any token:
https://crypto-api.info/api/{token-contract-address}
example: https://www.crypto-api.info/api/0xEd1199093b1aBd0...
get circulating supply from a token:
crypto-api.info/circulating/{query-name}
example: https://www.crypto-api.info/circulating/hex2t
add circulating supply for a token to crypto-api.info
