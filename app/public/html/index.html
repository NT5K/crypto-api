crypto-api.info

get public api for any token:
https://crypto-api.info/api/{token-contract-address}
example: https://www.crypto-api.info/api/0x514910771af9c...
get circulating supply from a token:
crypto-api.info/circulating/{token-contract-address}
example: https://www.crypto-api.info/circulating/0x514910771af9c...
add circulating supply for your token to crypto-api.info
add custom api for your token (coming soon)
